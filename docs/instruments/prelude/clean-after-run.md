# Prelude Clean After Run

This is a routine cleaning procedure to perform after a synthesis run. It is not a full [System Clean](system-clean.md), but still effective for day-to-day maintenance.

## Procedure

1. **Replace Glass RVs with Plastic RVs**
    - Navigate to **Operations > Cleaning > Wash RVs**
    - Run the wash cycle twice for good measure

2. **Remove reagent solutions** (optional)
    - Empty any amino acid solutions, deprotection solutions, coupling solutions, etc. if you want to save them
    - If you do not need to save them, skip this step

3. **Backflush all used bottles** (except DCM and DMF)
    - Go to **Bottle Preparations** and select every bottle you used (excluding DCM and DMF)
    - Run **Solvent Backflush** twice
    - Run **Nitrogen Backflush** twice

4. **Clean bottles**
    - Remove all dirty bottles from the Prelude and rinse with acetone
