# Prelude System Clean

A full system cleaning procedure for the Prelude peptide synthesizer.

## Before the System Clean

1. **Wash RVs**
    - Swap out the glass reaction vials (RVs) for plastic RVs
    - Navigate to **Operations > Cleaning > Wash RVs**

2. **Empty all solvent bottles** (except DCM)
    - Use only the wax-coated bottles
    - Make sure each solvent line is connected to a bottle
    - Have all amino acid bottles on as well

3. **Prime and pressurize the DCM bottle**

4. **Start the System Clean**
    - Navigate to **Operations > Cleaning > System Clean**

## After the System Clean

1. **Dump DCM** out of all bottles (amino acid and solvent), except the DCM bottle itself, and rinse with acetone

2. **Wash frits and solvent lines** (the lines and the small white piece at the end of each line) with acetone
