# Solution-Phase Methods

Protocols for solution-phase transformations commonly used in cyclic peptide synthesis, including macrolactamization, amide coupling, and classical organic reactions.

## Protocols

- [Cyclization](cyclization.md) —Low-concentration macrolactamization for head-to-tail cyclic peptides.
- [Amide Coupling with DSC](amide-coupling-dsc.md) —Disuccinimidyl carbonate-mediated amide bond formation.
- [Schotten-Baumann Reaction](schotten-baumann.md) —Classical acylation reaction for installing carbamate protecting groups.
