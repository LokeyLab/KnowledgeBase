Steps:
1. Wash RVs

  • Make sure you swap out the glass reaction vials (RVs) for plastic RVs
  • Click Operations -> Cleaning -> Wash RVs

2. Make sure all solvent bottles are empty (only use the wax coated bottles) except DCM.

  • Make sure each solvent line is connected to a bottle
  • Have all amino acid bottles on as well

3. Prime and Pressurize DCM Bottle
4. Start System Clean

  • Operations -> Cleaning -> System Clean

After System Clean is Done:
1. Dump DCM out of all bottles (amino acid and solvent) (except DCM bottle), rinse with acetone
2. Wash Fritz + solvent lines (lines and little white thing at the end of each line) with acetone
