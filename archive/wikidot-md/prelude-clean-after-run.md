# Prelude Clean After Run
NOT a System Clean But Still Pretty Good

1. Replace Glass RVs (Reaction Vials) with Plastic RVs
   - Operations -> Cleaning -> Wash RVs
   - Do it twice for good measure

2. Make sure to empty any amino acid solutions, deprotection solutions, coupling solutions, etc. if you want to save them, if not, skip this step.

3. Go to Bottle Preparations and select every single bottle you used (except DCM and DMF)
   - Solvent Backflush twice
   - Nitrogen Backflush Twice

4. Take all dirty bottles off prelude and rinse with Acetone
